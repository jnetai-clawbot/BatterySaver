# Battery Saver - Change Log

## v1.0.0 - Initial Release

### Project Overview
Battery Saver is an Android app that monitors battery health in real-time. It alerts users with sound and vibration when:
- Battery level drops below a user-configurable threshold (default 10%)
- Battery temperature exceeds safe limits (default 45°C)
- Unstable charging voltage or current is detected

The app runs as a background foreground service and auto-starts on boot for continuous protection.

### Key Features Implemented
1. Real-time battery monitoring (level, temperature, voltage, charging status)
2. Configurable low battery alert threshold (1-30%, default 10%)
3. Configurable overheat alert threshold (30-60°C, default 45°C)
4. Unstable power detection (over/under voltage, battery health failures)
5. Sound alarm with AUDIBILITY_ENFORCED flag (works even on mute)
6. Vibration alerts
7. Foreground service with persistent notification
8. Auto-start on boot (configurable)
9. Dark centered theme with Material Design
10. About section with "Made by jnetai.com", version number, Check for Updates, Share App
11. Three-tab interface: Monitor, Settings, About
12. Alert dismiss via notification action or app dialog
